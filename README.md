# LeagueCustomELO
A project that can be used to track how players play within custom games, giving each player an ELO and changing it based on custom game results in League of Legends.


In no way is this project making an MMR/ELO checker/estimator, attempting to make an alternative to Riot's ranked system, or even acessing any league player's ranks/ranked games to do so. This project is not affiliated with Riot Games whatsoever.
